# Visão geral do Adobe [!DNL Firefly Graph]

Se você já usou uma ferramenta de IA generativa antes, provavelmente está acostumado a digitar um prompt, obter um resultado e começar do zero na próxima vez que precisar de algo semelhante. O Firefly gráfico funciona de forma diferente. Em vez de um único prompt, crie um gráfico: um fluxo de trabalho visual, passo a passo, no qual todas as entradas, transformações e saídas estejam conectadas e salvas juntas.

>[!TIP]
>
>Pense nisso como uma receita ao invés de uma única refeição. Depois de criar um gráfico, você pode entregá-lo, reexecutá-lo em novas entradas ou alterar uma etapa e deixar que tudo seja atualizado automaticamente.

Esta página mostra tudo o que você precisa para criar seu primeiro gráfico: os conceitos principais, o primeiro fluxo de trabalho passo a passo e uma biblioteca de modelos prontos para abrir e adaptar imediatamente.
