# &#x200B;5. Biblioteca de modelos

Um índice de referência rápida de modelos de Firefly, organizado pelo que cada um produz ou faz. Cada exemplo é um ponto de partida — troque sua própria marca, produto e prompts antes de usar um modelo na produção.

## Geração e estilo da imagem

| Modelo de gráfico | Descrição | Casos de uso |
|---|---|---|
| [**Introdução - Gerar uma imagem**](/help/firefly/graph/templates/get-started-gen-image.md) | Um gráfico básico com um nó de prompt em um nó de geração em uma saída. | <ul><li>Imagem principal</li><li>Teste de espaço reservado</li><li>Imagem de amostra</li></ul> |
| [**Geração consistente de caracteres**](/help/firefly/graph/templates/character-gen.md) | Carregue uma imagem de referência de um personagem e, em seguida, troque a cena ou o prompt de pose para cada nova tomada. | <ul><li>Mascote recorrente</li><li>Personagem Spokes</li><li>Personagem do professor</li></ul> |
| [**Extração de estilo**](/help/firefly/graph/templates/style-extraction.md) | Alimente uma imagem de referência para extrair sua cor, luz e tratamento de textura. | <ul><li>Transferência de aparência</li><li>Correspondência de aparência sazonal</li><li>Correspondência de humor</li></ul> |
| [**Vibrações do pôr do sol**](/help/firefly/graph/templates/sunset-vibes.md) | Crie uma imagem tipográfica 3D usando um prompt de texto. | <ul><li>Sobreposição de slogan</li><li>Slogan da estação</li><li>Ativo de venda de Flash</li></ul> |
