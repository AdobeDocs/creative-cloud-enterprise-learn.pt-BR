# &#x200B;4. Personalizar um modelo

Cada modelo é fornecido com entradas de amostra e um caso de uso genérico. Ele se torna um fluxo de trabalho que funciona para *você* apenas quando reflete seu próprio resumo, ativos e resultados.

1. **Substitua as entradas de amostra por seu próprio ativo ou resumo.**
2. **Renomeie qualquer nó ou rótulo que ainda use texto de espaço reservado.**
3. **Execute o gráfico uma vez, completamente, antes de confiar no resultado.**
4. **Altere uma variável e execute-a novamente** para que você possa ver exatamente o que essa etapa controla.
5. **Salve sua versão personalizada com seu próprio nome.**
