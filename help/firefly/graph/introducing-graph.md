# Preenchimento generativo

Saiba como usar o Preenchimento generativo para adicionar, expandir ou remover conteúdo de imagens de forma não destrutiva usando prompts de texto simples. O preenchimento generativo permite aprimorar sua criatividade e criar projetos extraordinários. Experimente o Adobe Firefly [aqui](https://firefly.adobe.com/?locale=br).
