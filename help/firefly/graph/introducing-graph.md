# Apresentando o Firefly Graph

Neste webinar, saiba mais sobre o gráfico de sistema de fluxo de trabalho visual Firefly, projetado para ajudar você a compartilhar fluxos de trabalho entre suas equipes. O gráfico permite que qualquer pessoa aproveite seu fluxo de trabalho e gere ativos de qualidade de produção em uma fração do tempo.
