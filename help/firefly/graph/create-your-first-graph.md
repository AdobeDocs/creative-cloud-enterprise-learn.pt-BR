# Crie seu primeiro gráfico

Assim que souber o que são um nó, uma conexão e um modelo, você estará pronto para criar seu primeiro fluxo de trabalho.

1. **Abra uma tela em branco ou inicie a partir de um modelo.** Navegue na biblioteca de nós ou na galeria de modelos para encontrar um ponto de partida que esteja próximo do que você precisa.
2. **Conecte nós para criar seu fluxo de trabalho.** Desenhe conexões da saída de um nó para a entrada do próximo nó para encadear etapas juntas — por exemplo, uma imagem de entrada fluindo para um nó de estilo, depois para um gerador de variações, depois para a exportação.
3. **Adicione controles de entrada para tudo o que você deseja alterar mais tarde.** Transforme valores estáticos em controles deslizantes, seletores de cores ou campos de texto para que você (ou um colega de equipe) possa ajustar os parâmetros principais sem editar o gráfico subjacente.
4. **Execute o gráfico e visualize cada etapa.** Você pode visualizar a saída em qualquer nó antes que o fluxo de trabalho completo termine de ser executado.
5. **Altere uma entrada e execute novamente.** Troque uma imagem de referência, ajuste uma solicitação de estilo ou ajuste uma variável, e tudo será atualizado automaticamente. Nada é reconstruído à mão.

## O que experimentar primeiro

Se você não souber por onde começar, abra um dos modelos prontos na biblioteca de modelos em vez de uma tela em branco. Essa é a maneira mais rápida de ver um gráfico completo e funcional antes de criar um do zero.
