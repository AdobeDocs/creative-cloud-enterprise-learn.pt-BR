# Design de brinquedo de vinil

Saiba como inserir uma referência de personagem ou mascote e renderizá-la em uma forma de brinquedo de vinil estilizada. Há ângulos de rotação para uma plataforma de licenciamento ou revisão de produtos. [Abrir modelo de design de brinquedo de vinil](https://firefly.adobe.com/graph/edit/id/urn:aaid:sc:US:6b1e062a-bf16-5dc9-99dd-c3bd4061e238).

[!BADGE Exemplos do setor]{type=Informative tooltip="Exemplos do setor"}

* **Varejo** - Crie uma edição limitada de colecionador vinculada a um lançamento do programa de fidelidade, revisada como um conceito antes de se comprometer com uma execução de fabricação.
* **Bebidas** - Simule uma figura colecionável de um mascote da marca para uma entrega de mercadorias de execução limitada.
* **Entretenimento** - Visualize um personagem em formato de brinquedo de vinil para uma apresentação de licenciamento.

>[!TIP]
>
>**Antes de começar** - Para obter os melhores resultados, personalize este modelo de acordo com sua própria marca, produto e fluxo de trabalho. Troque suas imagens de referência, prompts e copie antes de usar qualquer saída.
